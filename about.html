About This Project

The Interactive Mental Health Check-in Board is a web-based application designed to help users track their daily moods and access supportive mental health resources. Built using HTML, CSS, and JavaScript, this project serves as both a personal mood journal and a gateway to professional mental health support.

Mental health awareness is an area I'm deeply committed to, and I believe technology can play a crucial role in breaking down barriers to support and self-care. This project represents my dedication to learning new technologies while contributing to important social causes.
